## 🔗 Integration with TellBill App

### CTA Button Flows
**All CTAs should point to**: `https://app.tellbill.com/signup?trial=true`

Buttons on landing page:
1. Hero "Start Free Trial" → App signup
2. Navigation "Start Free Trial" → App signup
3. Pricing "Start Free Trial" → App signup
4. Final CTA "Start Your Free 14-Day Trial" → App signup
5. Demo "Test TellBill Now" → Demo section or app

### Implementation
```javascript
// In scripts.js, replace trial signup handler:
function handleTrialSignup(e) {
    window.location.href = 'https://app.tellbill.com/signup?trial=true';
}
```

### Analytics Connection
Track conversions in TellBill app:
```javascript
// Google Tag Manager
gtag('event', 'landing_cta_clicked', {
    button_location: 'hero'
});
```
