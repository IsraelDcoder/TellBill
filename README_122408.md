TellBill – Setup & Run Guide


1. Repository Structure
/
├─ clients/        # Frontend (Expo / React Native)
├─ server/         # Backend (Node.js + Express)
├─ shared/         # Shared modules + DB schema
├─ migrations/     # Database migrations
├─ assets/
├─ .env.example
├─ docker-compose.yml
├─ package.json

2. Prerequisites

Node.js 18+

npm

PostgreSQL (local)

Git


4. Database Setup
Option A: Docker (recommended)
docker-compose up -d postgres

Option B: Local PostgreSQL

Create database:

createdb tellbill_dev

5. Run Migrations
npm run db:push


This sets up all tables.

6. Running the Backend

Backend lives in the server/ folder.

cd server
npm install
npm run dev


Backend runs on:

http://localhost:3000


Health check:

GET /api/health

7. Running the Frontend (Mobile App)

Frontend lives in clients/.

cd clients
npm install
npm run dev


Use Expo Go or emulator to open the app.

8. Common Issues
Backend not reachable from mobile

Ensure this matches:

EXPO_PUBLIC_BACKEND_URL=http://localhost:3000


Restart Expo after changing .env.

Database connection error

Check:

DATABASE_URL


Ensure PostgreSQL is running.

9. Summary

To run TellBill locally:

git clone <repo>
npm install
cp .env.example .env
docker-compose up -d postgres
npm run db:push
cd server && npm run dev
cd clients && npm run dev


Notes

Backend = server/

Frontend = clients/

Shared schema = shared/

.env.example must exist in repo
